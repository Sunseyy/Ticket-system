Etape 01:
Cette étape est nécessaire pour permettre la création automatique des PersistentVolumeClaims dans un cluster Kubernetes local (sans cloud provider).
--------------------------------------------------------------------------------------------------------------------------------
# Installation de storageclass local-path
kubectl apply -f https://raw.githubusercontent.com/rancher/local-path-provisioner/master/deploy/local-path-storage.yaml

# Verification des storageclass existant
kubectl get sc

# Retirer le statut "par défaut" de l'ancien StorageClass 
kubectl patch storageclass <nom-ancien-sc> -p '{"metadata": {"annotations":{"storageclass.kubernetes.io/is-default-class":"false"}}}'

#  Définir local-path comme StorageClass par défaut
kubectl patch storageclass local-path -p '{"metadata": {"annotations":{"storageclass.kubernetes.io/is-default-class":"true"}}}'

# Verification
kubectl get sc
